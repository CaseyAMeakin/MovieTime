MovieTime! is a website that helps people discover and explore movies, writers, actors, and directors.

Discovery is powered by a collaborative filtering algorithm to pair users and critics.

It is an MVC based site built on django, MySql, and Bootstrap, developed by Casey Meakin as a side project.

Some of the source code can be found at github:CaseyAMeakin.
